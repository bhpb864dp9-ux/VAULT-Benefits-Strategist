# NOTE ON ARCHITECTURE
# ─────────────────────────────────────────────────────────────────────────────
#
# VAULT is designed as a ZERO-DEPENDENCY static web application.
# The only "server" requirement is the ability to serve static files.
#
# Serving options:
#   1. Python:  python -m http.server 8080
#   2. Node.js: npx serve src/main/public
#   3. Docker:  docker run -p 80:80 vault-v1
#   4. nginx:   Copy files to /usr/share/nginx/html
#   5. Apache:  Copy files to /var/www/html
#   6. S3/CloudFront: Upload to S3 bucket
#   7. Netlify/Vercel: Deploy directly from Git
#
# This architecture ensures:
#   - No server-side vulnerabilities
#   - No database to secure
#   - No API keys to protect
#   - Minimal hosting costs
#   - Maximum deployment flexibility
